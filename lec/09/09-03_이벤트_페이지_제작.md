# 이벤트 페이지 제작

하나의 페이지를 만드는 실습  
이전의 것은 바디만 전송하는 반면  
레이어 팝업 혹은 창을 새로 띄울수도 있음.  
html과 CSS를 한 페이지로 작업  

이미지의 설명이 많으면 longdesk 코드로 짜고 연결해 주기도 한다.  
longdesk 안쓰고 그림을 CSS로 넣고 텍스트를 안보이도록 하는 행태로 작업함

dt : 제목, dd : 내용
태그만으로 약간 차이가 나도록 꾸며봄

dt 자식 요소로는 inline 요소가 올수 있음.
strong, em, span, a 가능
p, h 등은 올수 없음.

dd 안에는 모두 가능함.
